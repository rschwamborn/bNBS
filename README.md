# new method: bNBSS 

Data and R scripts for the paper: "Is normalized biomass really abundance? Pitfalls, artifacts, and misconceptions in the field of size spectra analysis - with a case for the use of back-transformed spectra (bNBSS)" 


Data and R scripts for bNBSS (new NBSS method), "back-transformed  and normalized numbers size spectrum" (Biomass vs body mass spectrum, normalized, redimensionalized, and  rescaled), with synthetic and in situ example data.
